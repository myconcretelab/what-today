Museo / Museo Slab fonts in use
--------------------------------

This folder already contains OTF binaries that are referenced by
`../fonts.css`:

- Museo300-Regular.otf  -> family 'Museo' weight 300
- Museo700-Regular.otf  -> family 'Museo' weight 700
- Museo_Slab_700.otf    -> family 'Museo Slab' weight 700

If you add other weights/styles (e.g., Museo_Slab_300.otf), remember to
declare a matching @font-face block in `../fonts.css`.

Note: Museo and Museo Slab are licensed fonts. Ensure you have rights to embed them.
